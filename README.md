# VaulDeSecretosBlazorServerEFSqlite

## pasos para iniciar el proyecto
 1. Descargar e instalar dotnet core <https://dotnet.microsoft.com/download> 
 2. dotnet restore
 3. dotnet ef database update
 4. dotnet run
